# Retargeting Playbook

## 1. Retargeting Strategy Framework

### 1.1 Retargeting Fundamentals

```
Retargeting Overview
====================

WHAT IS RETARGETING:
-------------------
Retargeting (also called remarketing) is a digital marketing strategy that 
allows you to show targeted advertisements to users who have previously 
visited your website, used your mobile app, or interacted with your brand 
in some way. It works by placing a tracking pixel or code on your digital 
properties, which then allows you to serve ads to these users as they browse 
other websites and platforms.

WHY RETARGETING MATTERS:
-----------------------
- Users typically need 7-12 touchpoints before converting
- Only 2-5% of visitors convert on first visit
- Retargeting typically has 10x higher conversion rates than cold traffic
- Cost per acquisition is often 50-70% lower than prospecting
- Allows for personalized messaging based on user behavior

KEY RETARGETING PLATFORMS:
--------------------------
1. Google Ads Remarketing (Display & Search)
2. Meta Ads (Facebook & Instagram)
3. LinkedIn Remarketing
4. Twitter/X Remarketing
5. Programmatic/DSP Platforms
6. Native Advertising Platforms
7. Email-based retargeting (via CRM)

RETARGETING TYPES:
------------------

Pixel-Based Retargeting:
- Uses tracking pixels placed on website
- Cookies users across the web
- Most common and widely available
- Examples: Google Ads, Meta, Display networks

List-Based Retargeting:
- Uses customer/visitor lists uploaded to platforms
- Matched to user profiles
- Allows for longer window periods
- Examples: LinkedIn, some DSP platforms

Search Retargeting:
- Targets users based on search behavior
- Reaches users on display inventory
- Based on keyword searches
- Examples: Google Customer Match, some DSPs

Social Retargeting:
- Targets users who engaged on social platforms
- Engagements include: page visits, video views, post engagement
- Examples: Meta, LinkedIn, Twitter

Email Retargeting:
- Uploads email lists for cross-device targeting
- Matches emails to user profiles
- Can be combined with pixel data
- Examples: LiveRamp, Facebook Custom Audiences
```

### 2.2 Segment Implementation Guide

```
Segment Creation & Implementation
==================================

GOOGLE ADS REMARKETING:
=======================

Standard Remarketing:
1. Install Google Ads remarketing tag on all pages
2. Create audience lists based on:
   - All visitors
   - Specific page visits
   - Time on site
   - Pages per session
3. Apply audience targeting to Display campaigns
4. Set frequency caps
5. Exclude converted users

Steps to Create:
1. Sign in to Google Ads
2. Navigate to "Audiences" in shared library
3. Click "Create audience"
4. Define audience rules
5. Name and save audience
6. Wait for audience to populate (usually 2-14 days)

Audience Rule Examples:
- Visited specific page: URL contains "/pricing"
- Visited multiple pages: Includes page "/products" AND "/demo"
- Time on site: Spent more than 2 minutes
- Pages visited: Viewed more than 3 pages

META ADS (FACEBOOK/INSTAGRAM):
==============================

Pixel-Based Audiences:
1. Install Meta Pixel on website
2. Create Custom Audiences based on:
   - Website visitors (by time)
   - Page visitors
   - Action takers
3. Create Lookalike Audiences

Steps to Create:
1. Go to Audiences in Ads Manager
2. Click "Create Audience"
3. Select "Custom Audience"
4. Choose "Website" as source
5. Define audience parameters
6. Set audience size and name
7. Create

Audience Parameters:
- All website visitors (7-180 days)
- Visitors by page (specific URLs)
- Visitors by time spent
- Video viewers (50%, 75%, 95%)
- Lead form submitters
- Checkout initiators

Lookalike Creation:
1. Select source Custom Audience (best customers)
2. Choose country(s) for lookalike
3. Select similarity percentage (1-10%)
4. Create lookalike audience

Similarity Tiers:
- 1% = Most similar, smallest reach
- 2-3% = Good balance
- 5-10% = Broader reach, less similar


LINKEDIN ADS:
=============

Matched Audiences:
1. Upload customer email list
2. Install LinkedIn Insight Tag
3. Create company page followers audience

Steps to Create:
1. Navigate to Campaign Manager
2. Select "Audiences" under "Plan"
3. Click "Create audience"
4. Select audience type
5. Upload list or configure Insight Tag rules
6. Save audience

Audience Types:
- Contact List Upload (email)
- Company Page Followers
- Website Visitors (via Insight Tag)
- Group Members

Website Visitor Rules:
- All visitors (last 30-180 days)
- Specific page visitors
- Job title matches
- Company size matches


PROGRAMMATIC/DSP:
=================

Common DSP Audiences:
1. First-Party Data (your own)
2. Third-Party Data (provider)
3. Contextual Data (page content)

First-Party Segments:
- Site visitors
- Customer lists
- App users
- Email subscribers

Third-Party Segments:

## 3. Frequency Optimization

### 3.1 Frequency Management Strategy

```
Frequency Optimization Framework
================================

WHY FREQUENCY MATTERS:
======================
- Under-frequency: Users forget your brand
- Optimal frequency: Brand stays top of mind
- Over-frequency: Ad fatigue, negative sentiment
- Critical balance: Maximize impact without irritation

INDUSTRY BENCHMARKS:
====================

Display/Programmatic:
- Optimal: 5-8 impressions per week
- Under-frequency: Under 3 per week
- Over-frequency: 15+ per week

Social Media (Facebook/Instagram):
- Optimal: 3-5 impressions per week
- Under-frequency: Under 2 per week
- Over-frequency: 10+ per week

Search Retargeting:
- Optimal: 4-6 impressions per week
- Under-frequency: Under 3 per week
- Over-frequency: 12+ per week

Video Ads:
- Optimal: 2-3 completions per week
- Under-frequency: Under 1 per week
- Over-frequency: 5+ per week


FREQUENCY CAPPING STRATEGIES:
==============================

PLATFORM-LEVEL CAPS:
--------------------

Google Display:
- Daily frequency cap: 3-5 impressions
- Weekly frequency cap: 10-15 impressions
- Campaign frequency cap: 30-50 impressions

Meta Ads:
- Frequency cap: 2-4 impressions per week
- Frequency goal: Set bid strategy focus
- Campaign budget: Let algorithm optimize

LinkedIn:
- Daily frequency cap: 3-5 impressions
- Weekly: 10-15 impressions

Twitter/X:
- Daily frequency cap: 5-10 impressions
- Weekly: 20-30 impressions

TIERED FREQUENCY STRATEGY:
---------------------------

Tier 1 - New Audiences:
- Cap: 1-2 impressions per day
- Goal: Build awareness without irritation
- Duration: First 14 days

Tier 2 - Active Engagement:
- Cap: 2-4 impressions per day
- Goal: Reinforce message while fresh
- Duration: Days 15-30

Tier 3 - Conversion Push:
- Cap: 4-6 impressions per day
- Goal: Drive action with urgency
- Duration: Days 30-45

Tier 4 - Fatigue Prevention:
- Cap: 1-2 impressions per day
- Goal: Maintain presence without fatigue
- Duration: Day 45+ or as needed


FREQUENCY BY AUDIENCE SEGMENT:
==============================

Segment                  | Frequency Cap | Duration
-------------------------|---------------|----------
New Visitors             | 2-3/day       | 7 days
Product Viewers          | 3-4/day       | 14 days
Pricing Visitors         | 4-6/day       | 14 days
Cart Abandoners          | 6-8/day       | 7 days
Past Visitors (30 days)   | 1-2/day       | 30 days
Past Visitors (60+ days)  | 1/day         | 30 days
Active Customers          | 2-3/day       | 30 days
```

### 3.2 Ad Fatigue Management

```
Ad Fatigue Prevention & Management
===================================

FATIGUE INDICATORS:
===================

Performance Metrics:
- CTR decline: -30% or more
- Conversion rate decline: -20% or more
- Frequency up, engagement down
- Cost per conversion increase: +25%+

Engagement Metrics:
- Like/comment/share decline
- Video completion rate drop
- Negative feedback increase

IDENTIFYING FATIGUE:
====================

Weekly Review Triggers:
□ CTR dropped significantly week-over-week
□ Frequency > 8 impressions with no conversion
□ Same creative running 3+ weeks without refresh
□ Audience size depleting faster than normal

Campaign Health Signals:
- Impressions stable, clicks dropping
- Frequency rising, conversions flat
- CPM increasing without reason
- Quality Score declining

FATIGUE PREVENTION STRATEGIES:
===============================

STRATEGY 1: CREATIVE ROTATION
------------------------------
- Refresh creative every 2-3 weeks
- Keep 3-5 active variations per campaign
- A/B test new creatives against old
- Set rotation rules in platform

Rotation Cadence:
Week 1-2: Creative A + B active
Week 3: Add Creative C, pause lowest performer
Week 4-5: Creative B + C active
Week 6: Add Creative D, evaluate A
Continue cycle

STRATEGY 2: AUDIENCE REFRESH
-----------------------------
- Expand audience when depleted
- Add lookalike audiences
- Create new audience segments
- Exclude converted users
- Split test new audience layers

STRATEGY 3: MESSAGE VARIATION
-----------------------------
- Rotate value propositions
- Change call-to-action focus
- Test different offers
- Vary emotional appeal
- Update proof points

STRATEGY 4: FREQUENCY MANAGEMENT
---------------------------------
- Lower daily caps
- Exclude recent converters
- Extend audience window
- Pause when fatigue detected
- Reset with fresh creative

FATIGUE RESPONSE PROTOCOL:
===========================

Step 1: Identify (Weekly Analysis)
- Review performance metrics
- Check frequency levels
- Assess creative age

Step 2: Diagnose (Root Cause)
- Is it creative fatigue?
- Is it audience exhaustion?
- Is it frequency too high?
- Is it competitive pressure?

Step 3: Act (Immediate Response)
- Pause underperforming creative
- Increase frequency cap if too low
- Expand audience if depleted
- Test new creative immediately

Step 4: Optimize (Ongoing)
- Monitor new creative performance
- Adjust frequency as needed
- Plan next creative refresh
- Document learnings

FATIGUE-FREE CAMPAIGN STRUCTURE:
=================================

Best Practice Structure:
- Campaign: Objective + Audience type
- Ad Set 1: Audience segment + Creative A
- Ad Set 2: Same audience + Creative B
- Ad Set 3: Lookalike + Creative A
- Ad Set 4: Lookalike + Creative C

Rotation Logic:
- Each ad set has 2-3 creatives
- Auto-optimization enabled
- Minimum 5,000 impressions per creative
- Weekly performance review
- Monthly creative refresh

CREATIVE FATIGUE TRACKER:
==========================

Template:

Review Date: [Date]

Creative Performance:

│ Creative   │ Impr.  │ CTR    │ Conv%  │ Fatigue? │
├─────────────┼────────┼────────┼────────┼──────────┤
│ Creative A │ [X]    │ [X.X]% │ [X.X]% │ [Y/N]    │
│ Creative B │ [X]    │ [X.X]% │ [X.X]% │ [Y/N]    │
│ Creative C │ [X]    │ [X.X]% │ [X.X]% │ [Y/N]    │
└─────────────┴────────┴────────┴────────┴──────────┘

Action Items:

2. [Creative to test next]
3. [New creative needed?]

## 4. Advanced Retargeting Strategies

### 4.1 Sequential Retargeting

```
Sequential Retargeting Playbook
================================

WHAT IS SEQUENTIAL RETARGETING:
==============================
Sequential retargeting delivers a series of ads to users in a specific order, 
guiding them through a journey from awareness to conversion. Each ad builds 
on the previous message, creating a narrative that increases engagement and 
conversion likelihood.

WHY SEQUENTIAL WORKS:
====================
- Mirrors natural buying journey
- Prevents message overload
- Builds brand story progressively
- Increases emotional investment
- Reduces ad fatigue through variety

SEQUENTIAL FRAMEWORK:
====================

SEQUENCE 1: AWARENESS TO CONVERSION
------------------------------------

Ad 1 - Hook (Day 1):
Objective: Capture attention
Message: Problem or insight
Format: Broad appeal, engaging

Ad 2 - Solution Introduction (Day 3-5):
Objective: Present solution
Message: How you solve the problem
Format: More detailed, benefit-focused

Ad 3 - Proof (Day 7-10):
Objective: Build credibility
Message: Social proof, results
Format: Testimonial or case study

Ad 4 - Call to Action (Day 12-15):
Objective: Drive conversion
Message: Clear offer, urgency
Format: Direct response

SEQUENCE 2: CART ABANDONMENT RECOVERY
--------------------------------------

Ad 1 - Reminder (1 hour):
Objective: Recall abandoned item
Message: "You left something behind"
Creative: Product image + name

Ad 2 - Benefit Reinforcement (6 hours):
Objective: Address objections
Message: Why this product is right
Creative: Benefits + reviews

Ad 3 - Urgency (24 hours):
Objective: Create urgency
Message: Limited availability
Creative: Stock warning + offer

Ad 4 - Final Push (48 hours):
Objective: Last conversion attempt
Message: Special incentive
Creative: Discount or bonus

SEQUENCE 3: RE-ENGAGEMENT WIN-BACK
------------------------------------

Ad 1 - What's New (Week 1):
Objective: Reintroduce brand
Message: New features, updates
Format: Newsletter-style content

Ad 2 - Value Reminder (Week 2):
Objective: Recall benefits
Message: Why they chose you
Creative: Past purchase + appreciation

Ad 3 - Special Offer (Week 3):
Objective: Drive re-engagement
Message: Exclusive win-back offer
Creative: Strong discount

Ad 4 - Last Chance (Week 4):
Objective: Final attempt
Message: Ending soon + loyalty appeal
Creative: Urgency + appreciation


IMPLEMENTATION TIPS:
====================

Frequency Management:
- Limit to 1-2 ads per week in sequence
- Don't rush the sequence
- Allow time between messages
- Cap total exposure in sequence

Creative Consistency:
- Maintain visual brand identity
- Reference previous ad when relevant
- Build narrative progression
- Match format to message stage

Attribution Considerations:
- Track conversion path by sequence
- Compare sequential vs. non-sequential
- Measure assisted conversions
- Optimize sequence order based on data

Platform Requirements:
- Facebook: Use carousel or video series
- Google: Use sequential audiences
- Programmatic: Use frequency caps
- LinkedIn: Personalize by persona
```

BEST PRACTICES:

1. Enable cross-device tracking in all platforms

4. Create device-specific landing pages
5. Optimize bids based on device conversion rates
6. Test device-sequential messaging
7. Review cross-device attribution reports regularly

Dynamic retargeting automatically shows users ads featuring the specific

   - Upload product catalog

2. Pixel Integration:
   - Track product page views
   - Track add to cart
   - Track purchase events

3. Ad Generation:
   - System creates dynamic ads
   - Automatically shows viewed products
   - Personalizes messaging

4. Bidding & Optimization:
   - Automated bid optimization
   - Time-based bidding
   - Audience-based bidding


DYNAMIC AD TYPES:
=================

-------------------------

E-commerce:
- Show products user viewed

- Show solutions/case studies

- Customized messaging

TRAVEL:
- Show searched destinations
- Dynamic pricing
- Availability indicators

REAL ESTATE:
- Show viewed properties
- Neighborhood info
- Price and features

AUTOMOTIVE:
- Show viewed vehicles
- Specifications
- Dealer location

## 5. Retargeting Measurement & Attribution

### 5.1 Retargeting Analytics Framework

```
Retargeting Analytics Framework
=================================

KEY METRICS FOR RETARGETING:
=============================

Awareness Metrics:
- Reach: Number of unique users targeted
- Frequency: Average impressions per user
- CPM: Cost per 1,000 impressions
- Brand lift: Incremental awareness

Engagement Metrics:
- CTR: Click-through rate
- Engagement rate: Social interactions
- Video completion rate: For video ads
- Time on site: Post-click behavior

Conversion Metrics:
- CPA: Cost per acquisition
- Conversion rate: Percentage converting
- ROAS: Return on ad spend
- Assisted conversions: Influence on path

Efficiency Metrics:
- Cost per reach point: CPM ÷ Reach
- Engagement cost: Cost per engagement
- Recovery rate: % of abandoners converted
- Customer acquisition cost: Total cost ÷ customers

RETARGETING-SPECIFIC KPIs:
==========================

Audience Health Metrics:
- Audience size: Total in segment
- Audience depletion rate: Speed of audience drying
- Segment freshness: Days since last visit
- Cross-segment users: Users in multiple segments

Frequency Metrics:
- Effective frequency: Optimal touchpoints for conversion
- Fatigue threshold: Frequency where performance drops
- Frequency distribution: % of users at each level

Attribution Metrics:
- View-through conversions: Pixel sees but no click
- Click-through conversions: Direct clicks
- Attribution window: Time considered for credit
- Assisted vs. last-click ratio

ROI Metrics:
- Retargeting ROAS: Revenue ÷ Ad spend
- Incremental lift: Retargeting vs. no retargeting
- Halo effect: Impact on non-retargeted conversions
- Customer lifetime value: For retargeted customers


ATTRIBUTION MODELS FOR RETARGETING:
====================================

LAST-CLICK ATTRIBUTION:
-----------------------
Pros:
- Simple to implement
- Easy to understand
- Direct correlation to revenue

Cons:
- Undervalues awareness touchpoints
- Ignores retargeting influence
- Can over-credit final ads

Recommendation: Use for tactical optimization

FIRST-CLICK ATTRIBUTION:
-------------------------
Pros:
- Shows true acquisition drivers
- Credits initial engagement
- Values prospecting

Cons:
- Ignores later influence
- Complex user journeys misattributed
- Difficult for short-term optimization

Recommendation: Use for strategic planning

LINEAR ATTRIBUTION:
-------------------
Pros:
- Equal credit to all touchpoints
- Simple model
- Fair representation

Cons:
- Doesn't account for impact
- All touchpoints equal regardless of importance
- Can inflate cold traffic value

Recommendation: Use for balanced view

TIME-DECAY ATTRIBUTION:
-----------------------
Pros:
- Recent touchpoints weighted more
- Reflects actual influence
- Good for consideration campaigns

Cons:
- Can undervalue awareness
- Requires sufficient data
- Complex model

Recommendation: Use for consideration campaigns

POSITION-BASED (U-SHAPED):
--------------------------
Pros:
- Credits first and last
- Reflects funnel importance
- Balances awareness and conversion

Cons:
- Middle touchpoints under-valued
- Arbitrary percentages
- Not flexible for all journeys

Recommendation: Use for standard attribution

DATA-DRIVEN ATTRIBUTION:
------------------------
Pros:
- Algorithm-based credit
- Accounts for actual impact
- Most accurate representation

Cons:
- Requires large data volume
- Complex to implement
- May need platform support

Recommendation: Use when available and data supports
```

### 6.2 Common Mistakes to Avoid

```
Retargeting Mistakes & Solutions
=================================

MISTAKE 1: OVER-FREQUENCY
--------------------------
Problem: Users see ads too often, causing fatigue and negative sentiment

Signs:
- CTR declining over time
- Negative comments increasing
- Audience reporting issues
- CPL increasing despite volume

Solutions:
- Set strict frequency caps
- Rotate creative frequently
- Exclude recent converters
- Reduce daily caps
- Expand audience segments

Best Practice:
- Start with low frequency caps
- Increase only if under-performing
- Monitor engagement decline
- Refresh creative every 2-3 weeks


MISTAKE 2: POOR AUDIENCE SEGMENTATION
--------------------------------------
Problem: Treating all visitors the same, leading to irrelevant messaging

Signs:
- Low CTR across all campaigns
- High bounce rate on landing pages
- No progression through funnel
- Conversion rate flat

Solutions:
- Create granular segments
- Match message to stage
- Personalize creative by segment
- Use behavioral data
- Implement sequential targeting

Best Practice:
- Segment by engagement level
- Target by page depth
- Personalize by interest
- Match offer to intent


MISTAKE 3: GENERIC CREATIVE
----------------------------
Problem: Using same ads for retargeting as prospecting

Signs:
- Low engagement from warm audiences
- High frequency needed for impact
- No differentiation from competitors
- Brand recall low

Solutions:
- Create retargeting-specific creative
- Reference their previous visit
- Show products they viewed
- Use urgency and exclusivity
- Include social proof

Best Practice:
- Dynamic product ads
- Personalized messaging
- Clear value proposition
- Strong CTAs


MISTAKE 4: IGNORING ATTRIBUTION
---------------------------------
Problem: Not understanding retargeting's true impact

Signs:
- Only tracking last-click
- Retargeting shows low ROAS
- Budget cut despite good performance
- Can't prove value

Solutions:
- Implement multi-touch attribution
- Track assisted conversions
- Measure lift vs. control group
- Calculate halo effect
- Report full funnel impact

Best Practice:
- Test with holdout groups
- Measure incremental lift
- Report influenced revenue
- Calculate true ROAS including assisted


MISTAKE 5: POOR LANDING PAGES
-------------------------------
Problem: Sending retargeted traffic to generic or homepage

Signs:
- High landing page bounce rate
- Low time on page
- Conversion rate not improving
- High cost per conversion

Solutions:
- Create dedicated landing pages
- Match ad message to page content
- Show previously viewed products
- Reduce friction
- Optimize for mobile

Best Practice:
- Dynamic landing pages
- Pre-populate forms
- Show social proof
- Clear CTAs
- Fast load times


MISTAKE 6: LONG RETARGETING WINDOWS
------------------------------------
Problem: Targeting users who visited long ago with no intent

Signs:
- Large cold audiences
- Low engagement rates
- Rising CPA over time
- Poor conversion rates

Solutions:
- Shorten retargeting windows
- Focus on recent visitors
- Use time-decay weighting
- Segment by recency
- Quality over quantity

Best Practice:
- Core audience: 7-14 days
- Extended: 30 days max
- Consider 60-90 day for high-consideration
- Review by segment performance


MISTAKE 7: NOT EXCLUDING CONVERTERS
-------------------------------------
Problem: Continuing to show ads to users who already converted

Signs:
- Wasted spend on existing customers
- Negative feedback from customers
- Customer complaints about ads
- Inflated frequency numbers

Solutions:
- Implement conversion exclusion
- Set up pixel-based exclusions
- Use customer match lists
- Regular list refresh
- Platform exclusion tools

Best Practice:
- Exclude at pixel level
- Upload customer email lists
- Regular sync with CRM
- Separate campaigns for customers
- Upsell vs. new acquisition
```
