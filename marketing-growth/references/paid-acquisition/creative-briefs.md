## 5. Creative Best Practices

### 5.1 Platform-Specific Guidelines

```
Platform Creative Best Practices
================================

FACEBOOK/INSTAGRAM:
====================

DO:
- Lead with the hook in first 3 seconds
- Use high-contrast visuals
- Keep text minimal on images
- Test vertical (4:5) for more real estate
- Include captions in videos (80% muted)
- Use authentic, relatable imagery
- Show real people when possible
- Match ad tone to audience expectation

DON'T:
- Use overly promotional tone
- Clutter with too much text
- Use low-quality imagery
- Mislead with clickbait
- Overuse emojis
- Ignore mobile-first design
- Neglect brand consistency

GOOGLE SEARCH:
==============

DO:
- Include primary keyword in headline
- Match user intent in copy
- Use numbers and statistics
- Include trust signals
- Have clear, specific CTAs
- Match landing page message
- Test multiple variations
- Address objections proactively

DON'T:
- Use vague or generic copy
- Overuse ALL CAPS
- Make false claims
- Neglect negative keywords
- Use competitor names without permission
- Ignore ad extensions

LINKEDIN:
=========

DO:
- Lead with professional value proposition
- Reference industry-specific insights
- Use authoritative tone
- Include relevant data points
- Target copy to seniority level
- Use professional imagery
- Focus on business outcomes
- Reference thought leadership

DON'T:
- Use casual or overly salesy language
- Use stock photos that look generic
- Make vague promises
- Ignore industry nuances
- Focus on features over benefits
- Use outdated statistics

YOUTUBE:
========

DO:
- Create strong hook in first 5 seconds
- Deliver on the promise immediately
- Keep middle engaging with pacing
- Use captions and graphics
- Match video quality to category
- Include clear end screen CTA
- Optimize for sound-off viewing
- Consider skippable vs non-skippable

DON'T:
- Bury the message
- Use slow pacing
- Rely on talking heads only
- Make the ad too long for the value
- Use intrusive music
- Neglect mobile viewing
- Ignore category norms

TIKTOK:
=======

DO:
- Embrace native, platform-specific style
- Use trending sounds and formats
- Start with hook in first 1-2 seconds
- Keep content fast-paced
- Show authentic, real content
- Reference trends when relevant
- Use text overlays effectively
- Be bold and creative

DON'T:
- Copy TV commercial style
- Use overly polished/corporate content
- Ignore trending sounds
- Use vertical video shot horizontally
- Be too salesy
- Underestimate Gen Z/millennial audience
- Use misleading content
- Post without understanding community norms
```

### 5.2 Creative Testing Framework

```
Creative Testing Framework
==========================

TESTING HIERARCHY:
==================

HIGH IMPACT TESTS:
1. Ad Format (Image vs. Video vs. Carousel)
2. Value Proposition Framing
3. Hook/Opening Strategy
4. Audience-Creative Match

MEDIUM IMPACT TESTS:
5. CTA Button Selection
6. Offer/Discount Presentation
7. Social Proof Type
8. Visual Style/Treatment

LOW IMPACT TESTS:
9. Color Variations
10. Minor Copy Adjustments
11. Logo Placement
12. Aspect Ratio Optimization

TEST DESIGN BEST PRACTICES:
===========================

SAMPLE SIZE CALCULATION:
- Minimum: 1,000 impressions per variation
- Recommended: 5,000-10,000 impressions
- Statistical significance: 95% confidence level

TEST DURATION:
- Search: 2-4 weeks minimum
- Social: 1-2 weeks minimum
- Display: 4-6 weeks minimum

Test until:
- Statistical significance achieved
- Minimum runtime completed
- Clear winner identified

VARIABLE ISOLATION:
-------------------
Test one variable at a time:
✓ Test Hook A vs. Hook B (same visual)
✗ Test Hook A + Visual A vs. Hook B + Visual B

If testing multiple variables:
- Use multivariate testing (MVT)
- Requires larger sample sizes
- Use factorial design

DOCUMENTATION:
--------------
Document for each test:
- Hypothesis
- Test setup (control vs. variant)
- Audience targeting
- Test duration
- Results (before and after)
- Statistical significance
- Learnings
- Recommendations

TEST RESULTS TEMPLATE:
=====================

Test Name: [Name]
Date Range: [Start] - [End]
Duration: [X] days

Control Version:
- Description: [What was tested]
- Performance: [Metrics]

Variant Version:
- Description: [What was tested]
- Performance: [Metrics]

Results:
┌──────────┬──────────┬──────────┬──────────┐
│ Metric   │ Control  │ Variant  │ Change   │
```
