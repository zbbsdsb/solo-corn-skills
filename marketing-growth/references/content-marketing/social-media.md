# Social Media Platform Setup Guides

## 1. Major Social Media Platforms Overview

### 1.1 Instagram (Business Account)

```
Instagram Business Setup
=========================

PLATFORM OVERVIEW:
- Type: Visual-first social platform
- Audience: 2+ billion monthly active users
- Best for: B2C, lifestyle, fashion, food, travel, e-commerce

BUSINESS ACCOUNT SETUP:
Step 1: Convert to Business Profile
1. Go to Settings → Account
2. Tap "Switch to Professional Account"
3. Choose "Business" category
4. Complete business information

Step 2: Connect to Facebook Page
1. Link Instagram to Facebook Page
2. Enable cross-platform messaging
3. Set up unified inbox

Step 3: Set Up Instagram Shopping
1. Convert to Business or Creator account
2. Connect Facebook Catalog
3. Submit for Instagram Shopping review
4. Enable shopping in posts and stories

Step 4: Configure Messaging Settings
1. Set up automated responses
2. Create saved replies
3. Enable away messages
4. Configure Quick Replies

Step 5: Analytics Setup
1. Access Instagram Insights
2. Set up branded content tracking
3. Configure story and reel analytics
4. Set up notifications for key metrics

BEST PRACTICES:
- Post consistently (3-5 times per week)
- Use Reels for reach and discovery
- Engage within 1 hour of posting
- Use all interactive features (polls, quizzes, Q&A)
- Collaborate with creators
```

```
TikTok Business Setup

3. Business hours
4. Contact information

Step 3: Set Up TikTok Ads Manager
1. Create business ads account
2. Set up pixel for website tracking
3. Configure conversion events
4. Set up billing

Step 4: Content Strategy Setup
1. Identify trending sounds
2. Create content calendar
3. Set up hashtag tracking
4. Monitor For You Page (FYP) performance

Step 5: Analytics Configuration
1. Access TikTok Analytics
2. Set up content performance tracking
3. Monitor follower growth
4. Track engagement rates

BEST PRACTICES:
- Post daily or multiple times per day
- Keep videos under 15 seconds for engagement
- Use trending sounds and hashtags
- Engage in comments within 1 hour
- Duet and Stitch other creators
- Create original audio
```

### 1.5 Snapchat (Business)

```
Snapchat Business Setup
=======================

PLATFORM OVERVIEW:
- Type: Ephemeral messaging platform
- Audience: 750+ million monthly active users
- Best for: Gen Z targeting, app installs, AR experiences

BUSINESS ACCOUNT SETUP:
Step 1: Create Business Account

2. Create Snapchat account

2. Create AR lenses
3. Design branded filters
4. Set up Sponsored Geofilters

BEST PRACTICES:
- Post to Story daily
- Use Snap Map features
- Create AR experiences
- Partner with Snap Stars (influencers)
- Use lens for brand awareness
- Track Swipe Up rates
```

## 3. Unified Social Media Strategy

### 3.1 Cross-Platform Content Adaptation

```
Content Adaptation Matrix
=========================

CONTENT TYPE           INSTAGRAM    TIKTOK     LINKEDIN    FACEBOOK    SNAPCHAT    PINTEREST
─────────────          ─────────    ──────     ────────    ────────    ─────────    ─────────
Behind-the-scenes      Reels        Videos     Articles    Stories     Snaps        Board
Product Launch         Carousel     Video      Post+Poll   Video+CTA   AR Lens      Pins
Educational            Carousel     Tutorial   Article     Video       ✗            Infographic
User Testimonial       Reel         Clip      Recommendation  Video     ✗            ✗
Industry News          Story        Video     Article     Post        ✗            ✗
How-To Guide           Carousel     Step-video   Doc      Video       ✗            Tutorial Pin
Team Spotlight         Grid Post    Video      Post        Album       Snap Coll    ✗
Promo/Coupon           Story+Post   Video     Direct Msg  Video+CTA   Snap         Product Pin
Event Coverage         Story+Lives  Recap     Live        Live+Story  Live         Board
Q&A/Survey             Story Poll   Comment   Poll        Poll        Poll         Pin

ADAPTATION RULES:
-----------------
1. Video: Square (1:1) for IG Feed, Vertical (9:16) for TikTok/Snapchat/Stories
2. Text: 125-150 words for FB/LinkedIn, Short captions for IG/TikTok
3. Hashtags: 1-2 for LinkedIn, 5-10 for IG, 3-5 for Twitter
4. Images: High-res for Pinterest, Optimized crops for each platform
5. Links: UTM parameters for all platforms, Trackable URLs
```

### 3.3 Analytics Integration

```
Cross-Platform Analytics Tracking
=================================

UNIFIED TRACKING SETUP:
1. UTM Parameters for All Links
   - utm_source: Platform name
   - utm_medium: Social media
   - utm_campaign: Campaign name
   - utm_content: Specific content piece

2. Platform-Specific Pixels/Tags
   - Facebook Pixel
   - Instagram Insights (built-in)
   - TikTok Pixel
   - LinkedIn Insight Tag
   - Pinterest Tag
   - Snapchat Pixel

3. Google Analytics Integration
   - Social traffic attribution
   - Goal completions from social
   - User flow from social channels

4. Third-Party Analytics Tools
   Recommended tools:
   - Metricool (all-in-one)
   - Sprout Social (enterprise)
   - Buffer Analyze (scheduling-focused)
   - Hootsuite Analytics (team management)

KEY METRICS TO TRACK:
Platform          Primary Metrics

Instagram         Reach, Saves, Shares, Engagement Rate, Follower Growth

Pinterest          Repins, Clicks, Saves, Follower Growth
```
