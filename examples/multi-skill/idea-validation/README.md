# Idea Validation - Idea Validation Combination Example

## Overview
Combines Landing, Mental Models, and Product Builder Skills to complete the journey from idea to validation.

## Skill Combination
- **Landing**: Idea clarification and constraint definition
- **Mental Models**: First principles and inversion thinking analysis
- **Product Builder**: MVP definition and validation plan

## Usage
```bash
node index.js
```

## Workflow
1. Use Landing to clarify idea and constraints
2. Use mental models to challenge assumptions
3. Use ProductBuilder to design validation-oriented MVP
4. Execute validation plan

## Example Code
```javascript
const IdeaValidation = require('./index');
const validator = new IdeaValidation('Your idea');
validator.validate();
```
