## 🎯 How to Use

4. Understand how Skills interact with each other

## 💡 Combination Tips

- Skills can be used in series, with one Skill's output serving as the next Skill's input
- Skills can also be used in parallel, with multiple Skills working simultaneously
- Pay attention to data format compatibility between Skills
