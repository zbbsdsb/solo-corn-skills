# Strategic Execution - Strategic Execution Combination Example

## Overview
Combines Strategic Decision, Mental Models, and Marketing Growth Skills to complete the cycle from strategy to execution.

## Skill Combination
- **Strategic Decision**: SWOT, market sizing analysis
- **Mental Models**: Second order thinking, probabilistic thinking, scenario planning
- **Marketing Growth**: Audience, positioning, growth loops

## Usage
```bash
node index.js
```

## Workflow
1. Strategic analysis (SWOT, market sizing)
2. Deep thinking (second order thinking, scenario planning)
3. Growth planning (audience, positioning, growth loops)

## Example Code
```javascript
const StrategicExecution = require('./index');
const execution = new StrategicExecution('Your business idea');
execution.execute();
```
