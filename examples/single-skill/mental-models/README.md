# Mental Models - Thinking Framework Examples

## Overview
Demonstrates how to use various mental models for decision-making and thinking.

## Included Models
- **First Principles**: Start from fundamental truths, challenge assumptions
- **Inversion**: Think about how to fail, then avoid it
- **Second Order Thinking**: Consider ripple effects
- **Circle of Competence**: Recognize knowledge boundaries
- **Probabilistic Thinking**: Evaluate multiple possibilities

## Usage
```bash
node index.js
```

## Example Code
```javascript
const MentalModels = require('./index');
const mm = new MentalModels();

mm.firstPrinciples('How to succeed in entrepreneurship', [
  'Need large amounts of capital',
  'Fundamental: Solve real problems'
]);
```
