# Code Examples Repository

A comprehensive collection of examples demonstrating the SOLO-Corn-Skills framework capabilities, from single skill usage to complete workflows.

## 📚 Examples Overview

This repository contains **36 example files** organized into **5 categories** to help you quickly understand and use the framework.

## 🚀 Quick Start

Navigate to the examples directory:
```bash
cd examples
```

Run any example:
```bash
cd <category>/<example-name>
node index.js
```
