# Project Templates

This directory contains reusable project templates to help you quickly get started with SOLO CORN SKILLS.

## 📋 Template List

| Template | Description |
|----------|-------------|
| [**basic-starter/**](basic-starter/) | Minimalist basic project skeleton, suitable for quickly setting up new projects |
| [**saas-starter/**](saas-starter/) | Complete SaaS project with frontend-backend separation architecture |

## 🎯 Quick Start

### Basic Starter

```bash
cd basic-starter
npm install
npm start
```

### SaaS Starter

```bash
cd saas-starter
npm install
npm start
# Open browser at http://localhost:3000
```

## 💡 How to Use

1. Select the appropriate template directory
2. Copy it to your workspace
3. Modify configuration and code as needed
4. Start developing!
