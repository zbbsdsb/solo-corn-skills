# ASCII Interface Template Library

This library provides ready-to-use ASCII templates for rapid prototyping. Copy, customize, and adapt these templates to your specific needs.

## Quick Reference

| Template Type | Use Case | Time to Create |
|---------------|----------|----------------|
| Form | User input collection | 15-30 min |
| List | Data display and management | 10-20 min |
| Dashboard | Multi-metric overview | 20-40 min |
| Navigation | Site/app structure | 5-10 min |

## Customization Guidelines

### Character Set Reference

```
Box Drawing:
┌ ─ ┐ │ └ ┘ ┘ ╭ ╮ ╰ ╯ ╱ ╲ ║
─────────────────────────────────────
Borders: + - | ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼
Arrows: ← → ↑ ↓ ↔ ⇐ ⇒ ⇑ ⇓ ⇔
```

### Common Adaptations

1. **Adjust width**: Modify character count within lines
2. **Add columns**: Extend table headers and dividers
3. **Change labels**: Replace placeholder text with your content
4. **Add rows**: Copy existing row structure and modify
5. **Combine templates**: Nest smaller templates into larger layouts
