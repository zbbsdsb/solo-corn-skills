# Evidence Collection Framework

Building genuine faith requires systematic evidence collection. This framework provides structured methods for gathering, evaluating, and integrating evidence to support well-founded beliefs.

## 1. Evidence Categories

### 1.1 Supporting Evidence (Pro-Evidence)

Evidence that strengthens or confirms a belief:

| Category | Description | Examples |
|----------|-------------|----------|
| **Direct Evidence** | Firsthand observations or measurements | Experimental results, direct measurements, field observations |
| **Indirect Evidence** | Circumstantial evidence that implies truth | Correlations, patterns, logical deductions |
| **Testimonial Evidence** | Reports from credible sources | Expert opinions, documented experiences, peer validations |
| **Analogous Evidence** | Evidence from similar cases | Case studies, historical parallels, analogical reasoning |
| **Predictive Success** | Correct predictions based on the belief | Forecasts that come true, models that work |

### 1.2 Counter-Evidence (Contra-Evidence)

Evidence that challenges or contradicts a belief:

| Category | Description | Examples |
|----------|-------------|----------|
| **Direct Contradictions** | Evidence that directly opposes the belief | Failed experiments, observational anomalies |
| **Alternative Explanations** | Other plausible explanations for the same evidence | Confounding variables, alternative hypotheses |
| **Anomalies** | Unexplained phenomena within the framework | Edge cases, exceptions, outliers |
| **Negative Evidence** | Absence of expected evidence | Missing confirmations, null results |
| **Expert Dissents** | Credible disagreement from domain experts | Peer criticism, alternative theories |

## 2. Evidence Strength Assessment

### 2.1 Strength Evaluation Matrix

Assess each piece of evidence on multiple dimensions:

| Dimension | Score 1 (Weak) | Score 3 (Moderate) | Score 5 (Strong) |
|-----------|----------------|-------------------|------------------|
| **Reliability** | Unverified source, anecdotal | Partially verified, inconsistent | Independently verified, consistent |
| **Relevance** | Tangential connection | Moderate relevance | Directly addresses the belief |
| **Specificity** | Very general, vague | Somewhat specific | Highly specific, precise |
| **Reproducibility** | Non-reproducible | Partially reproducible | Consistently reproducible |
| **Objectivity** | Highly subjective | Mixed subjectivity/objectivity | Highly objective |

### 2.2 Evidence Weight Calculation

```
Evidence Weight = (Reliability × 0.25) + (Relevance × 0.25) + 
                  (Specificity × 0.20) + (Reproducibility × 0.20) + 
                  (Objectivity × 0.10)
```

- **Weight Score 4.0-5.0**: High-weight evidence, major factor in belief formation
- **Weight Score 2.5-3.9**: Medium-weight evidence, contributes meaningfully
- **Weight Score 1.0-2.4**: Low-weight evidence, minimal impact

## 3. Evidence Collection Methods

### 3.1 Systematic Search Protocol

```markdown
1. DEFINE THE CLAIM
   - State the belief or claim explicitly
   - Identify key components that require evidence
   - Determine the standard of proof needed

2. EXHAUSTIVE SEARCH
   - Primary sources (original research, direct data)
   - Secondary sources (reviews, analyses)
   - Tertiary sources (summaries, educational materials)
   - Cross-reference with domain experts

3. DOCUMENT EVERYTHING
   - Record source, date, and access method
   - Note any selection bias in source choice
   - Document why certain sources were excluded

4. ACTIVE COUNTER-EVIDENCE SEARCH
   - Deliberately seek contradictory evidence
   - Use "devils advocate" methodology
   - Search for failed experiments and negative results
```

## 4. Evidence Integration Methods

### 4.1 Cross-Evidence Validation

```markdown
1. CONVERGENCE ANALYSIS
   - Identify evidence that points to the same conclusion
   - Assess independence of converging evidence
   - Look for convergent validation across different methods

2. DIVERGENCE ANALYSIS
   - Identify conflicting evidence
   - Assess quality of each piece of conflicting evidence
   - Determine if conflicts are superficial or fundamental

3. TRIANGULATION
   - Combine multiple independent evidence sources
   - Verify through different methodologies
   - Strengthen confidence through multiple approaches
```

### 4.3 Gap Analysis

```markdown
1. IDENTIFY REQUIRED EVIDENCE
   - What evidence would be necessary to confirm the belief?
   - What evidence would be necessary to refute the belief?
   - What evidence is currently missing?

2. ASSESS EVIDENCE GAPS
   - Are the gaps critical or peripheral?
   - Can gaps be filled with reasonable effort?
   - Do gaps represent fundamental uncertainty?

3. ADJUST BELIEF STRENGTH
   - Stronger belief if core gaps are filled
   - Weaker belief if critical gaps remain
   - Consider if belief should be withheld until gaps are addressed
```

## 5. Continuous Evidence Monitoring

### 5.1 Evidence Review Schedule

```markdown
- **Initial Assessment**: Comprehensive evidence collection at belief formation
- **Weekly Review**: Check for new evidence in active investigations
- **Monthly Review**: Reassess beliefs with accumulated evidence
- **Quarterly Review**: Major reassessment of belief foundations
- **Triggered Review**: Immediate review when significant new evidence emerges
```

### 5.2 Evidence Update Protocol

```markdown
When new evidence emerges:

1. CLASSIFY THE EVIDENCE
   - Does it support or contradict?
   - What is its weight?

2. ASSESS THE IMPACT
   - Does it fundamentally change the evidence balance?
   - Is it consistent with existing evidence?
   - Does it open new lines of inquiry?

3. UPDATE THE BELIEF
   - Adjust confidence level appropriately
   - Document the reason for change
   - Note if belief needs more fundamental revision

4. COMMUNICATE CHANGES
   - Update relevant stakeholders
   - Revise documentation
   - Note lessons learned
```

## 6. Building Genuine Confidence

### 6.1 Evidence-Based Faith Criteria

True confidence (faith) should be proportional to:

1. **Comprehensiveness**: All available relevant evidence has been considered
2. **Quality**: Evidence meets reliability and validity standards
3. **Balance**: Both supporting and contradicting evidence have been weighed
4. **Currency**: Evidence is up-to-date and actively maintained
5. **Transparency**: The evidence basis is documented and open to scrutiny

### 6.2 Red Flags Warning

Exercise caution when:

- Only supporting evidence is considered
- Counter-evidence is dismissed without proper evaluation
- Evidence quality is not assessed
- Belief predates the evidence gathering
- Confidence exceeds what the evidence supports
- Evidence is selectively chosen to confirm preexisting beliefs

---

**Remember**: Genuine faith is not blind belief, but confidence that has been tested against available evidence and found to be well-supported.
