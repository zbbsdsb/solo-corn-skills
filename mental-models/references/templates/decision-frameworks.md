## 2. Second-Order Thinking Framework

**Use when:** Evaluating decisions with significant consequences, or where short-term vs. long-term tradeoffs exist.

### Decision Context
```
Proposed Action: ________________________________

First-Order Effect (immediate outcome):
_________________________________________________
```

### Second-Order Analysis
```
If I take this action, then...
  → What happens next? _________________________
  → And then? __________________________________

If I DON'T take this action, then...
  → What happens instead? ______________________
  → And then? __________________________________
```

### Time Horizon Analysis
```
Short-term (days/weeks): ________________________
Mid-term (months): _____________________________
Long-term (years): _____________________________
```

### Second-Order Question Checklist
- [ ] Who benefits immediately? Who benefits later?
- [ ] What behaviors does this incentivize?
- [ ] What system am I embedded in, and how will it respond?
- [ ] What feedback loops exist?
- [ ] What are the third-order effects?

### Decision Matrix
```
                    1st Order    2nd Order    3rd Order
Action A:            [+]          [-]          [?]
Action B:            [-]          [+]          [+]
Action C:            [+]          [+]          [-]
```

## 4. Circle of Competence Decision Filter

**Use when:** Making career, investment, or strategic decisions.

### Self-Assessment
```
Decision Domain: ________________________________

My Knowledge Level: (1-10) ___________
My Experience Level: (1-10) __________
My Network/Resources: (1-10) __________

Competence Score: _______ (average)

Confidence Level: [High] [Medium] [Low]
```

### Knowledge Boundaries
```
What I Know (core competencies):
1. _____________________________________________
2. _____________________________________________
3. _____________________________________________

What I Know I Don't Know:
1. _____________________________________________
2. _____________________________________________

What I Don't Know I Don't Know (blind spots):
1. _____________________________________________
2. _____________________________________________
```

### Decision Pathway
```
Inside Circle of Competence:
→ Proceed with standard decision process
→ Trust your judgment

Outside Circle, Recognizable Gap:
→ Seek expert counsel
→ Allocate time for research
→ Consider partnerships

Outside Circle, Unknown Territory:
→ Acknowledge limitation
→ Use inversion (avoid mistakes)
→ Move slowly, test small
```

## 5. Bayesian Decision Framework

**Use when:** Updating beliefs based on new evidence, or quantifying uncertainty in decisions.

### Prior Beliefs
```
Hypothesis: ____________________________________

Prior Probability (before evidence): ____%

Confidence in prior: [High] [Medium] [Low]
```

### Evidence Assessment
```
New Evidence: __________________________________

Likelihood this evidence would occur IF hypothesis is true: ___%
Likelihood this evidence would occur IF hypothesis is false: ___%

Strength of Evidence: [Very Strong] [Strong] [Moderate] [Weak]
```

### Posterior Calculation
```
Base Rate (prior): ___%
Evidence Weight: ___%
Novelty Factor: ___%

Updated Probability (posterior): ___%

Belief Change: [Strengthened] [Unchanged] [Weakened]
```

### Action Threshold
```
Minimum threshold for action: ___%
Current confidence: ___%
Recommended Action: [Proceed] [Wait] [Abort]
```

Malice probability: ___%
```

### Final Interpretation
```

```

Regret level (1-10): _______

If I DON'T do this:
Regret level (1-10): _______
Duration of regret: _______________
```

### Probability-Weighted Regret
```
P(success): ___%
P(failure): ___%

Expected regret if action: (__% × ___ ) + (__% × ___ ) = ___
Expected regret if inaction: (__% × ___ ) + (__% × ___ ) = ___
```

### Decision
```

Emotional check: Can I live with this outcome?
[ ] Yes
[ ] No
[ ] Need more information
```

## 8. Reversibility Framework (Two Boxes)

**Use when:** Deciding whether to act quickly or gather more information.

### Decision Characteristics
```
Decision: _______________________________________

Reversibility: [Easily Reversible] [Partially Reversible] [Irreversible]

Cost to reverse: _________________________________

Time sensitivity: [Low] [Medium] [High]
```

### Reversibility Analysis
```
If REVERSIBLE:
→ Act quickly, learn from feedback
→ Make decision now: ____________________________

If IRREVERSIBLE:
→ Gather more information
→ Consult experts
→ Consider second-order effects
→ Decision timeline: ___________________________
```

### Speed vs. Accuracy Matrix
```
                    High Accuracy Needed
                    Yes              No
High Speed Needed   [Deliberate]    [Act Fast]
                    Slow            [Act Fast]
                    Deliberate
```

### Commitment Level
```
[ ] Trial/pilot possible
[ ] Phased commitment
[ ] Full commitment required
```

## 10. Compounding Decision Framework

**Use when:** Evaluating habits, systems, and long-term investments.

### Compounding Question
```
Does this decision compound over time?

[ ] Yes - positive compounding
[ ] Yes - negative compounding
[ ] Neutral - linear or one-time effect
```

### Compounding Analysis
```
For [Daily/Weekly/Monthly] action:

Time Period     Linear Impact    Compounding Impact
1 month         ___              ___
6 months        ___              ___
1 year          ___              ___
5 years         ___              ___
10 years        ___              ___
```

### Smallest Viable Action
```
Minimum effective action: ________________________
Frequency: _______________________________________
Time investment: ________________________________

Can I sustain this for 10 years? [Yes/No]
```

### Decision
```
High compounding, sustainable: → PRIORITIZE
High compounding, unsustainable: → REFINE or SKIP
Low/no compounding: → Evaluate efficiency
Negative compounding: → AVOID
```
