## 7. Problem Diagnosis Worksheet

**Purpose:** Avoiding common cognitive biases when diagnosing problems.

### Problem Statement
```

```

### Common Biases Check
```
Anchoring: Am I fixated on initial information?
[ ] Yes - initial data is dominating my thinking
[ ] No - I've gathered fresh data

Confirmation bias: Am I seeking evidence that confirms my initial hypothesis?
[ ] Yes - I need to actively seek disconfirming evidence
[ ] No - I've tested multiple hypotheses

Availability bias: Am I over-weighting recent or memorable events?
[ ] Yes - I should look at historical data
[ ] No - recent events are truly relevant

Survivorship bias: Am I only looking at successes?
[ ] Yes - I need to study failures
[ ] No - I've considered the full dataset
```
