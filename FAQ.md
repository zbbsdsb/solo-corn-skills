## General Questions

### Q: What is SOLO CORN SKILLS?

**A**: SOLO CORN SKILLS is an extensible AI skill framework designed for solopreneurs. It provides a collection of skills (mental models, research integration, product building, marketing growth, strategic decision making, etc.) that help you work smarter, make better decisions, and build businesses.

### Q: Is this software I install?

**A**: SOLO CORN SKILLS is primarily a documentation and reference framework. You don't "install" it in the traditional sense. You clone the repository and use the documentation and reference materials as guides. That said, the API documentation shows how the skills could be invoked programmatically if you build integrations.

### Q: Who is this for?

**A**: SOLO CORN SKILLS is designed for:
- Solopreneurs building businesses alone
- Founders making strategic decisions
- Developers building products
- Anyone who wants to improve their thinking and decision making
- People who want a structured approach to business and product development

### Q: Do I need to be a developer to use this?

**A**: No! While developers will find it useful, the project is designed to be accessible to everyone. The mental models, strategic frameworks, and documentation can be used by anyone regardless of technical background. The API documentation is there for those who want to build integrations, but it's not required to get value from the project.

### Q: What makes this different from other resources?

**A**: SOLO CORN SKILLS is unique because:
- It's a complete, integrated system (not just isolated concepts)
- Skills are designed to work together
- It's grounded in research and proven frameworks
- It's open source and community-driven
- It's specifically designed for solopreneurs and small teams

**A**: Follow these steps:

### Q: Do I need to use all the skills?

**A**: No! Use what's useful for your situation. You can use skills individually, or combine them for more powerful workflows. Start with what solves your most pressing problem, and explore others as needed.

## Skills & Usage

### Q: What are "skills" in SOLO CORN SKILLS?

**A**: Skills are modular capabilities that help you solve specific types of problems. Each skill includes:
- A SKILL.md file explaining what it's for and how to use it
- Reference materials in references/ directory
- Practical frameworks and templates
- Integration guides with other skills

### Q: How do I use a skill?

**A**: To use a skill:
1. Navigate to the skill's directory
2. Read the SKILL.md file
3. Explore the references/ materials
4. Apply the frameworks to your situation
5. Check [API.md](API.md) for programmatic usage patterns

### Q: Can I use multiple skills together?

**A**: Absolutely! In fact, that's where much of the power comes from. See [TUTORIALS.md](TUTORIALS.md) for examples of skill combinations. Some common combinations:
- **Landing → Product Builder → Marketing Growth** for building products
- **Collector → Mental Models → Strategic Decision** for making decisions
- **Connector → Research Integration → Mental Models** for learning

### Q: How do I know which skill to use?

**A**: The **Collector** skill (collector/) is designed to help with this! It helps you think about what skills would be most useful for your situation. In general:
- Think → Mental Models
- Decide → Strategic Decision
- Build → Product Builder
- Grow → Marketing Growth
- Learn → Research Integration, Connector
- Validate → Landing

## Mental Models

### Q: What are mental models?

**A**: Mental models are thinking frameworks that help you understand the world, make decisions, and solve problems. They're like lenses that help you see things more clearly. Instead of reinventing thinking patterns each time, you can use models that have been proven effective.

### Q: How many mental models are there?

**A**: SOLO CORN SKILLS includes **45+ mental models** across categories:
- Decision & Reasoning
- Strategy & Competition
- Systems & Complexity
- Psychology & Behavior
- Execution & Learning

See [mental-models/SKILL.md](mental-models/SKILL.md) for the complete list.

### Q: Do I need to learn all the mental models?

**A**: No! Start with a few that seem most relevant to your life and work. Get good at using them, then add more over time. Even 3-5 well-understood models can dramatically improve your thinking.

### Q: Which mental models should I start with?

**A**: Great starting points:
1. **First Principles** - Break problems down to basics
2. **Second-Order Thinking** - Think beyond immediate consequences
3. **Bayesian Thinking** - Update beliefs as you get new information
4. **Opportunity Cost** - What are you giving up?
5. **Inversion** - Think about what to avoid

### Q: How do I remember to use mental models?

**A**: It takes practice! Try these strategies:
- Pick 1-2 models and use them intentionally for a week
- Create a checklist or reminder to use models before decisions
- Review past decisions and see which models would have helped
- Keep a "thinking journal" where you document your reasoning

2. Open a new issue with details about what's confusing

### Q: Git is giving me errors - help!

**A**: Common Git issues and solutions:

**"Permission denied" when cloning**:
- Use HTTPS instead of SSH: `git clone https://github.com/zbbsdsb/solo-corn-skills.git`
- Or set up SSH keys properly

**"Not a git repository"**:
- Make sure you're in the right directory: `cd solo-corn-skills`
- Check if .git exists: `ls -la`

**Merge conflicts**:
- Don't panic! This is normal
- Read Git's documentation on resolving merge conflicts
- Ask for help if stuck

See [FORK.md](FORK.md) for more Git help.
