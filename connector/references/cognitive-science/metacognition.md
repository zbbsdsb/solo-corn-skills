# Metacognition

## Origin

**John Flavell** (1979), "Metacognition and Cognitive Monitoring," *American Psychologist*

## Core Concept

Metacognition = thinking about thinking = awareness and control of your own cognition.

### Two Components

1. **Metacognitive knowledge**: Understanding how your mind works
   - What are my strengths/weaknesses?
   - How do I learn best?
   - What strategies work for me?

2. **Metacognitive regulation**: Controlling your learning process
   - Planning: What will I learn?
   - Monitoring: How am I doing?
   - Evaluating: Did I succeed?

## Key Insight

> "Know what you know, and know what you don't know."

### Calibrated Uncertainty

Expert beginners know exactly what they don't know.
Novice beginners don't know what they don't know. (Dunning-Kruger)

## Application in Connector

### 1. Self-Assessment Checklists

**After QuickScan**:
```
□ I can define [domain] in one sentence
□ I can name 3 core concepts
□ I can identify 2 connections to my knowledge
□ I know what I still don't understand
□ I have questions for an expert
```

**After Learning Path**:
```
□ Surface layer: Fully understood ✓
□ Middle layer: Mostly understood, reviewing ⏳
□ Deep layer: Need more practice 🔄
□ Can explain to someone else
□ Can apply to my project
```

### 2. Learning Path Visualization

Show users WHERE they are in the learning journey.

```
[Domain Learning Map]

        Deep Expertise
           /    \
          /      \
    Middle ←──────→ [YOU ARE HERE]
          \      /

         Surface Entry
```

### 3. Blind Spot Identification

**Prompt**:
```
"You've learned [domain basics]. But what DON'T 
you know that you don't know? Here are common 
blind spots in [domain]..."
```

### 4. Strategy Selection

Different problems need different strategies.

**When to use what**:
| Situation | Strategy |
|-----------|----------|
| Don't know where to start | QuickScan first |
| Concepts confusing | Find analogies |
| Can't remember | Spaced repetition |
| Can't apply | Project-based learning |
| Talking to expert | Expert Prep module |

## Connector Implementation

### QuickScan: Metacognitive Check
```
"Before we begin, rate your familiarity:
1 = Never heard of it
2 = Heard the word
3 = Can explain basics
4 = Working knowledge
5 = Expert

This helps us calibrate the learning path."
```

### Learning Path: Regular Check-ins
```
"Checkpoint: What's your confidence level now?
What surprised you?
What's still confusing?
What do you want to dive deeper into?"
```

### Knowledge Archive: Reflection Prompts
```
"After each learning session:
1. What did I learn? (3 bullets)
2. How does this connect to what I know?
3. What questions remain?
4. What's my next action?"
```

## Key Takeaway

> "Learning how to learn is more important than learning itself."

## Research Benefits

Students who develop metacognitive skills:
- **Learn 30% faster**
- **Retain 50% more**
- **Transfer knowledge better**
- **Self-regulate effectively**

## Practical Metacognitive Questions

**Before Learning**
- What do I already know about this?
- What's my goal?
- How much time do I have?

**During Learning**
- Is this making sense?
- Should I try a different approach?
- What am I confused about?

**After Learning**
- Did I achieve my goal?
- What do I still need to learn?
- How can I apply this?
