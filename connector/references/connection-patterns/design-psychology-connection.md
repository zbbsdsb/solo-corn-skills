# Design-Psychology Connection Pattern

## Core Insight
Good design comes from a deep understanding of human psychology; psychological principles are the underlying logic of design decisions.

## Classic Connection Patterns

### 1. Cognitive Psychology → User Experience Design
#### Connection Points
- **Cognitive Load Theory** → **Information Layered Design**
- **Gestalt Principles** → **Visual Organization**
- **Working Memory Limitations** → **Interaction Simplification**

#### Applications
- **Cognitive Load**: Step-by-step guidance, only give necessary information at once
- **Gestalt**: Layout using proximity, similarity, continuity principles
- **Working Memory**: Avoid asking users to remember too much information

---

### 2. Behavioral Economics → Product Design
#### Design Applications of Psychological Effects
| Psychological Effect | Design Application |
|---------------------|-------------------|
| Anchoring Effect | Price display design |
| Loss Aversion | Emphasize losses of not using |
| Social Proof | User reviews display |
| Authority Effect | Expert recommendations |
| Scarcity | Limited-time offer design |

#### Examples
- Anchoring: Show high price first, then show discounted price
- Loss Aversion: "If you don't buy now, the price will increase"
- Social Proof: "Already chosen by 10,000 people"

---

### 3. Motivation Theory → User Engagement Design
#### Self-Determination Theory
- **Autonomy**: Give users choices
- **Competence**: Design difficulty gradients
- **Relatedness**: Community and social elements

#### Application Framework
1. Goal layering: From simple to difficult
2. Immediate feedback: Let users see progress
3. Social elements: Sharing and comparison
