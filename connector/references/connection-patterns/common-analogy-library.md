## Analogy Application Steps

1. **Identify**: Find core principles in source domain
2. **Map**: Map principles to target domain
3. **Validate**: Check if analogy applies
4. **Innovate**: Generate new ideas based on analogy
5. **Evaluate**: Test effectiveness of new ideas

⚠️ Analogies are not truth, just thinking tools
