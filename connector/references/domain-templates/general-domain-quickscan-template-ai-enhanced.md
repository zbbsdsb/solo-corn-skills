# General Domain QuickScan Template (AI-Enhanced)

## Template Description
This template is designed for quickly understanding any new domain, based on cognitive science theories of cognitive load and generation effect, and integrated with AI-assisted learning features.

## AI-Enhanced Features
- **Adaptive Content Synthesis**: AI automatically recommends the most relevant content based on your background and goals
- **Intelligent Knowledge Graph Generation**: Automatically discovers core concepts and relationships
- **Blind Spot Identification**: Analyzes your knowledge gaps

## Template Usage Instructions

This template is based on Connector Skill's cognitive science design principles:

| Cognitive Science Principle | Application in Template |
|------------------------------|-------------------------|
| **Cognitive Load Management** | Layered information disclosure, information chunking |
| **Generation Effect** | Active filling, prediction exercises, fill-in-the-blank |
| **Transfer Learning** | Forced search for cross-domain connections |
| **Metacognition** | Self-assessment and reflection |
| **Desirable Difficulties** | Moderate challenge, active retrieval |
| **Spaced Repetition** | Review reminders, long-term memory |
| **Situated Learning** | Real project application |

### Cognitive Load Checkpoints
- ✓ Only show 7±2 points per layer
- ✓ Use progressive disclosure
- ✓ Provide personalized analogies
- ✓ Allow pausing and digesting

### Generation Effect Checkpoints
- ✓ Active prediction instead of passive reading
- ✓ Fill-in-the-blank instead of looking at answers
- ✓ Active recall instead of re-reading
- ✓ Create notes instead of copying
