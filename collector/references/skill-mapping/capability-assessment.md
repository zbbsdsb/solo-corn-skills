# Skill Capability Assessment

This document provides comprehensive capability assessments for all skills in the Solo skill ecosystem, enabling accurate skill selection based on task requirements.

## Capability Assessment Framework

### Assessment Dimensions

Each skill is evaluated across 8 core dimensions:

| Dimension | Description | Scoring Range |
|-----------|-------------|---------------|
| **Depth** | Expertise level in core domain | 1-10 |
| **Breadth** | Range of covered topics | 1-10 |
| **Practicality** | Ready-to-use outputs | 1-10 |
| **Research Quality** | Academic/research rigor | 1-10 |
| **Implementation Support** | Code/artifact generation | 1-10 |
| **Speed** | Time to actionable output | 1-10 |
| **Learning Curve** | Ease of effective use (inverse) | 1-10 |
| **Scalability** | Handles increasing complexity | 1-10 |

## Usage Guidelines

### Skill Selection Based on Capability Match

1. **High Depth + Low Implementation** → Use for analysis only, then transition
2. **High Implementation + Low Analysis** → Use after strategic decisions made
3. **High Breadth + Specific Need** → Select skill with best coverage
4. **Time-Critical** → Choose skill with highest practicality score
